# Contract Define

Generador de contratos API compartidos entre frontend y backend a partir de RFC.

Shared API contract generator between frontend and backend from RFC.

## Description

Crea contratos API precisos que definen la interfaz compartida entre frontend y backend. Especifica modelos de datos, operaciones CRUD, eventos, flujos asincronos, y patrones de integración. Documenta expectativas claras de formato, validación, errores, y casos límite para evitar sorpresas durante implementación.

Creates precise API contracts that define the shared interface between frontend and backend. Specifies data models, CRUD operations, events, asynchronous flows, and integration patterns. Documents clear expectations for format, validation, errors, and edge cases to avoid surprises during implementation.

## SDD Phase

**F3 (Contract):** Defines the API contract phase where frontend and backend specifications align before implementation begins.
